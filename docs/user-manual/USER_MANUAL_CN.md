## 一、系统概述

### 1.1 什么是LinkC

LinkC是新一代智能清洁机器人管理平台，通过AI技术实现人机协同，帮助物业管理团队：

- **实时监控** - 随时掌握所有机器人的工作状态
- **智能调度** - AI自动优化任务分配
- **数据分析** - 用数据驱动决策优化
- **移动办公** - 手机随时处理工作事务

### 1.2 系统角色

| 角色 | 权限范围 | 主要使用功能 |
|------|----------|--------------|
| 管理员 | 全部功能 | 系统配置、用户管理 |
| 运营经理 | 监控+调度 | 战略驾驶舱、运营控制台 |
| 现场主管 | 任务+告警 | 任务管理、告警管理 |
| 操作员 | 移动端 | 移动监控、告警响应 |

### 1.3 系统访问

| 功能模块 | 访问地址 | 推荐设备 |
|----------|----------|----------|
| 战略驾驶舱 | /executive | 电脑/大屏 |
| 运营控制台 | /operations | 电脑 |
| 实时地图 | /demo/map | 电脑/大屏 |
| 任务管理 | /tasks | 电脑 |
| 告警管理 | /alerts | 电脑/手机 |
| 移动端 | /mobile | 手机/平板 |

**关键指标说明**：

- 周报：周度趋势分析

- 自定义：按需生成报表

## 四、运营控制台

### 4.1 运营总览 (/operations)

运营人员的主工作台，展示需要关注的实时信息。

**页面组成**：

1. **状态概览**
   - 在线机器人数量
   - 当前执行任务数
   - 待处理告警数
   - 排队等待任务数

2. **实时动态**
   - 最新告警通知
   - 任务状态变更
   - 机器人状态变化

3. **快捷操作**
   - 新建任务
   - 查看告警
   - 机器人控制

### 4.2 日常操作流程

**早班交接**：
1. 登录系统，查看运营总览
2. 检查是否有未处理的夜间告警
3. 确认所有机器人状态正常
4. 查看今日计划任务

**任务监控**：
1. 定期刷新实时地图
2. 关注任务进度
3. 及时响应告警
4. 记录异常情况

**晚班交接**：
1. 确认所有任务完成或已安排
2. 检查机器人充电状态
3. 处理完所有告警
4. 填写交接记录

1. 点击 [+] 放大

**热力图模式**：
1. 勾选"热力图"选项
2. 显示各区域清洁频率
3. 颜色越深表示清洁需求越高

## 六、任务管理

### 6.1 任务列表 (/tasks)

查看和管理所有清洁任务。

**任务状态**：
| 状态 | 说明 |
|------|------|
| 待执行 | 已创建，等待分配 |
| 进行中 | 机器人正在执行 |
| 已完成 | 任务成功完成 |
| 已取消 | 任务被取消 |
| 失败 | 执行失败，需重新安排 |

### 6.2 创建新任务

**步骤**：

1. 点击"新建任务"按钮
2. 填写任务信息：
   - 任务类型：日常清洁/深度清洁/消毒
   - 目标区域：选择楼层和区域
   - 优先级：普通/紧急
   - 计划时间：立即执行/定时执行
3. 点击"提交"
4. 系统自动分配最优机器人

**使用AI创建任务**：

1. 打开训练工作台 (/trainer)
2. 输入自然语言，如"安排大堂清洁"
3. AI分析并推荐方案
4. 确认后自动创建任务

### 6.3 任务操作

| 操作 | 说明 | 适用状态 |
|------|------|----------|
| 查看详情 | 查看任务完整信息 | 所有 |
| 修改 | 修改任务参数 | 待执行 |
| 取消 | 取消任务 | 待执行/进行中 |
| 暂停 | 暂停执行 | 进行中 |
| 恢复 | 恢复执行 | 已暂停 |
| 重新执行 | 重新分配执行 | 失败 |

## 七、告警管理

### 7.1 告警列表 (/alerts)

查看和处理系统告警。

**告警级别**：

| 级别 | 颜色 | 说明 | 响应要求 |
|------|------|------|----------|
| 紧急 | 红色 | 需立即处理 | 5分钟内响应 |
| 警告 | 橙色 | 需要关注 | 30分钟内处理 |
| 提示 | 黄色 | 一般提醒 | 当日处理 |
| 信息 | 蓝色 | 仅供参考 | 可忽略 |

### 7.2 常见告警类型

| 告警 | 说明 | 建议处理 |
|------|------|----------|
| 电量低 | 电量低于20% | 系统自动召回充电 |
| 遇到障碍 | 清洁路径被阻挡 | 现场移除障碍物 |
| 任务超时 | 任务执行超过预期 | 检查是否卡住 |
| 设备故障 | 硬件或软件异常 | 联系技术支持 |
| 离线 | 无法连接机器人 | 检查网络和电源 |

### 7.3 处理告警

**处理流程**：

1. 查看告警详情
2. 判断处理方式
3. 执行处理操作
4. 填写处理结果
5. 关闭告警

**处理操作选项**：
- 召回充电：让机器人返回充电
- 重启任务：重新执行当前任务
- 取消任务：取消当前任务
- 现场处理：标记需现场人员处理
- 忽略：标记为无需处理

## 九、AI智能助手

### 9.1 使用AI助手

在训练工作台 (/trainer) 可以使用自然语言与AI交互。

**可以问什么**：
- "现在有哪些机器人空闲？"
- "大堂的清洁进度如何？"
- "安排会议室B的紧急清洁"
- "把低电量的机器人都召回充电"

### 9.2 AI交互示例

**查询状态**：
```
用户: 现在机器人状态怎么样？
AI: 当前8台机器人状态如下：
    - 工作中: 4台 (A-01, A-02, A-05, A-07)
    - 空闲: 2台 (A-03, A-06)
    - 充电中: 1台 (A-04)
    - 故障: 1台 (A-08，建议检查)
```

**任务调度**：
```
用户: 安排大堂清洁
AI: 好的，我分析了当前情况：
    1. 大堂区域面积约500㎡
    2. 空闲机器人A-03距离最近，电量92%
    3. 预计清洁时间15分钟

    是否确认派遣A-03执行大堂清洁任务？
    [确认] [取消]
```

### 9.3 AI操作确认

涉及实际操作的AI建议需要人工确认：
- 任务创建/修改/取消
- 机器人召回/暂停
- 批量操作

## 十、常见问题

### 10.1 登录问题

**Q: 忘记密码怎么办？**
A: 联系系统管理员重置密码。

**Q: 提示"会话过期"？**
A: 重新登录即可，系统默认30分钟无操作自动退出。

### 10.2 地图问题

**Q: 地图上看不到机器人？**
A:
1. 检查是否选对楼层
2. 检查机器人是否在线
3. 刷新页面重试

**Q: 机器人位置不更新？**
A:
1. 检查页面右下角连接状态
2. 如显示"轮询模式"属正常降级
3. 如无响应请刷新页面

### 10.3 任务问题

**Q: 任务一直是"待执行"？**
A:
1. 检查是否有空闲机器人
2. 检查机器人电量是否充足
3. 检查目标区域是否可达

**Q: 任务执行失败？**
A:
1. 查看失败原因
2. 检查机器人状态
3. 必要时重新创建任务

### 10.4 告警问题

**Q: 收不到告警通知？**
A:
1. 检查浏览器通知权限
2. 检查手机通知设置
3. 确认账号有对应权限

**Q: 告警太多怎么办？**
A:
1. 可设置告警过滤条件
2. 可调整告警级别阈值
3. 联系管理员优化配置
